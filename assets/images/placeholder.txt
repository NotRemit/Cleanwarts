This folder will contain images for the application.

When you first run the application, you can use the "Generate House Emblems" button in development mode 
to create basic house emblems, or generate them using the canvas functions in app.js:

- gryffindor.png
- hufflepuff.png
- ravenclaw.png
- slytherin.png
- parchment-bg.jpg
- hogwarts-bg.jpg

You can replace these with your own custom images as needed.
